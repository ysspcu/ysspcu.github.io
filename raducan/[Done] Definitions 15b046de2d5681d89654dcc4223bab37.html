[Done] Definitions

LIST OF DEFINITIONS

API (Application Programming Interface)A set of tools and protocols that allow developers to create mods and interact with game’s code.
Modding APIA framework that allows developers to create mods that can interact with the game and other mods.
Mod or ModificationA user created add-on or alteration to the game that changes or enhances gameplay, graphics or other features.
Mod LoaderA tool that loads and manages mods, ensuring they are compatible with the game and each other.
Forge or Minecraft ForgeA popular modding framework/platform and API for Minecraft that allows developers to create and implement mods.
Fabric of Fabric LoaderAnother modding framework/platform and API for Minecraft, known for its lightweight and modular approach to modding
Risugami’s ModLoader or RMLThe very first modding API for Minecraft, providing very basic options, used to make mods, replaced by more complex and feature rich Forge.
Modding CommunityA group of players and developers who create, share, and use mods for a specific game.
ServerA multiplayer environment where players can connect and play Minecraft together, often with mods installed to enhance the experience.
PluginA type of mod specifically designed to add features or modify the gameplay on Minecraft Server.
Resource Pack or Texture PackA collection of files that changes game’s texture, sounds, and other visual elements without altering gameplay mechanics.
ModpackA collection of mods bundled together to create a specific gameplay experience, often curated and configured to work seamlessly together.
ShaderA mod that enhances the game’s graphics by adding advanced lightning, shadows, and other visual effects.
BiomeA specific type of environment in Minecraft, such as forests, deserts, oceans, each with its own unique features and resources.
ChunkA 16x16 horizontal area in Minecraft that is loaded and rendered by the game engine.
Java EditionThe original version of Minecraft, written in Java, which is the primary platform for most mods.
Bedrock EditionA version of Minecraft designed for cross-platform play, with a different codebase from Java Edition, and limited modding capabilities
Custom NPC (Non-Player Character)A character created by a modder to interact with players, often used to add quests, shops, or other interactive elements.
World GenerationThe process by which Minecraft creates its game world using unique seeds, including terrain, structures, and biomes often modified by mods to add new features.
SMP (Survival Multiplayer)A multiplayer mode in Minecraft where players survive and build together, often enhanced with mods.
Creative ModeA game mode in Minecraft that allows player unlimited resources and ability to fly, often used for building and testing mods.
Survival ModeThe default game mode in Minecraft where players must gather resources, build structures, and survive against monsters.
Adventure ModeA game mode designed for custom maps and adventures, where players can interact with objects and NPCs, but cannot break blocks.
Hardcore ModeA variant of Survival Mode with increased difficulty and permanent death, often used in challenge mods.
PatchAn update to a mod or the game itself that fixes bugs, add features, or improve performance.
CompatibilityThe ability of different mods to work together without conflicts and issues.
DependencyA mod or library that another mod requires to function correctly.
Config FileA file used to customize the settings and behavior of a mod.
MixinA tool for modifying the game’s code at runtime, allowing developers to inject custom behavior into existing classes without directly modifying the source code.
DecompilingThe process of converting compiled bytecode back into readable source code, often used in modding to understand and modify the game’s code.
ObfuscatingThe practice of making code difficult to understand on purpose, often used in Minecraft to protect the game’s source code. Modders must deobfuscate the code to create mods. [Wikipedia, Minecraft Modding]
JDK (Java Development Kit)A software development kit required for developing Java applications, including Minecraft Mods. [xx]
GradleA build automation tool used to manage dependencies and build processes in Minecraft modding projects.
JSON (JavaScript Object Notation)A lightweight data interchange format used in Minecraft for configuration files, such as defining custom items, blocks, and entities.
NBT (Named Binary Tag)A binary format used by Minecraft to store data, such as player inventories, world data, and entity information. [x]
CoremodA type of mod that makes fundamental changes to game’s core code, often using bytecode manipulation techniques.
Bytecode ManipulationThe process of modifying the compiled bytecode of a game to change its behavior, often used in coremods and mixins.
PatchouliA modding library used to create in-game documentation and guides, often used to provide instructions and information to players about a mod.
Data PackA collection of JSON files that modify the game's data-driven features, such as advancements, loot tables, and world generation settings.
Capability SystemA system in Minecraft Forge that allows mods to add custom data and behavior to entities, items, and other game objects.
Tile/Block EntityA special type of block entity in Minecraft that can store additional data and perform complex behaviors, often used for machines and storage blocks.
Packet HandlingThe process of sending and receiving data packets between the client and server, allowing for synchronization of game state and custom network communication.
Server-Side ModA mod that runs exclusively on the server, providing functionality and features that do not require client-side changes.
Client-Side ModA mod that runs exclusively on the client, providing visual and interface enhancements that do not affect the server.
Cross-Mod CompatibilityThe ability of different mods to work together without conflicts, often achieved through shared APIs and careful design.
Feed the BeastA collection of modpacks for Minecraft designed to enhance the game.
